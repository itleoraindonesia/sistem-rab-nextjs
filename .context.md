# Project Context

## Core Technologies
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: **shadcn/ui**
- **State**: TanStack Query (Server), Zustand (Client)
- **Backend**: Supabase

## UI & Styling Strategy
This project exclusively uses **shadcn/ui** for UI components.
- Components are located in `@/components/ui`.
- Styling is handled via Tailwind CSS classes.
- Theme customization is done in `globals.css`.
- Icons are strictly **Lucide React**.

## Documentation Workflow (Context7)
We use **context7** to verify documentation.
- **Rule**: Before implementing complex logic or using new component features, check the documentation via context7.
- **Scope**: Use context7 to check documentation for shadcn/ui, Next.js, and Supabase to ensure best practices and correct usage.

## Database
- **Source of Truth**: `src/types/database.ts` (Auto-generated from Supabase).
- Always sync types using `pnpm db:pull` when database schema changes.
