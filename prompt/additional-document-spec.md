1. Section 1: Identitas Surat
    - No (auto generated if approved)
    - Instansi (Dropdown :  PT Maju
    Mandiri Gemilang Terang, PT Leora
    Konstruksi Indonesia, dll)
    - Kategori Surat (text)
    - Tanggal (Automatic selected today)
2. Section 2: Konten Surat
    - Perihal. (change from “Judul surat”. short text)
    - Isi surat (long text)
3. Section 3: Pengirim (auto-filled dari login)
    - Pengirim (user login data. content : dept from, user name, user email)
4. Section 4: Penerima
    - Penerima (kolom nama instansi - short text, kolom nama
    penerima - short text, kolom alamat - short text , whatsapp number - phone number, email - email)
5. Section 5: Lampiran & Tanda Tangan
    - Lampiran dokumen (boolean ada/tidak. if yes, create multiple pdf/attach doc/file)
    - Kolom tanda tangan (signature section. can be multiple sign from user)
